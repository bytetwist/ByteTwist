superClass

fun superClass(): CompiledClass?

Returns the super class of this class, if the super class is one of the classes scanned. Returns null otherwise If you are trying to return just the name of the super class, use the ASM implementation of superName

Return
The CompiledClass object of the super class if the super class was one of the scanned classes, otherwise null
